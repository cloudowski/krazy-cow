  * healthcheck - mood decreaser
  * restartpolicy - /setfree

* secret - auth, tls certs

* deployment

* pv, pvc

  * resiliency - redis with pv, failure, proof
* envs / namespaces
  * separate ns with redis 
  * redis in cloud - configurable
